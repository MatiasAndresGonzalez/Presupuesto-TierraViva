Modularización

  • Técnico
  • Módulos
  • Presupuesto

  1. MODULO de Autenticación y Roles Entidad

    • Usuario: Login / Logout / Registro
    • Recuperar contraseña: Enviar mail con código
    • Rol: Crear rol, asignar permisos, editar permisos, eliminar rol

  2. MODULO de Usuarios (Clientes y Administradores)

    • Nombre, Email, Rol, Contraseña, Estado (activo/inactivo)
    • Registrar nuevo cliente o admin
    • Listar usuarios, buscar, filtrar por rol
    • Editar datos personales, cambiar rol
    • Baja lógica (no eliminación real, sólo inactivar)

  3. MODULO de Productos

    • Nombre, Categoría, Precio (minorista/mayorista), Descripción, Imágenes, Stock mínimo
    • Agregar nuevo producto
    • Ver listado con filtros (categoría, precio, stock bajo)
    • Editar precio, stock, descripción
    • Dar de baja el producto (lógica, no física)

  4. MODULO de Stock

    • Producto, Cantidad, Fecha de ingreso, Proveedor, Lote
    • Ingresar stock manual o por compra
    • Ver stock actual, buscar por producto o lote
    • Corregir stock manualmente (con motivo)
    • Eliminar entrada de stock (auditable)

  5. MODULO de Ventas

    • Cliente, Productos vendidos, Forma de pago, Total, Fecha, Estado
    • Registrar venta web o manual
    • Ver historial de ventas, filtros por cliente, fecha o producto
    • Actualizar estado de pedido (Ej: En preparación, Enviado, Entregado)
    • No se recomienda borrar ventas (solo anulación/estado cancelado)

  6. MODULO de Compras a Proveedores

    • Proveedor, Productos comprados, Fecha, Monto total
    • Registrar nueva compra
    • Consultar historial de compras por proveedor o fecha
    • Editar compra (solo antes del cierre mensual)
    • Eliminar compra (opcional, auditable)

  7. MODULO de Clientes

    • Nombre, CUIT/DNI, Email, Teléfono, Dirección, Estado de cuenta
    • Alta de nuevo cliente
    • Buscar cliente, ver historial, estado de cuenta
    • Editar datos personales
    • Baja lógica del cliente

  8. MODULO de Categorías de Producto

    • Nombre, Descripción, Activa/Inactiva
    • Crear categoría nueva
    • Listar categorías activas/inactivas
    • Editar nombre o descripción
    • Eliminar (baja lógica)

  9. MODULO de Notificaciones / Mensajes Internos

    • Tipo (mail/sistema), Destinatario, Contenido, Estado (leído/no leído)
    • Crear notificación (ej: pedido enviado)
    • Ver bandeja de entrada o historial de mensajes
    • Marcar como leído/no leído
    • Eliminar mensaje (opcional, solo para admin)
