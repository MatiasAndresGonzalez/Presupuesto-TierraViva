¿Qué se va a desarrollar?

El proyecto contempla el desarrollo de un sistema completo para la gestión de una distribuidora mayorista de productos artesanales, compuesto por dos partes principales:

  1. Tenda Online (eCommerce): un sitio web público donde los clientes mayoristas pueden navegar productos, hacer pedidos y consultar su historial.

  2. Sistema Interno de Gestión (ERP): una herramienta privada para uso interno de la empresa, que permite gestionar productos, stock, ventas, compras, clientes, reportes y automa zar tareas administra vas.

FASE 1 – Diseño UI/UX + Desarrollo Front-End eCommerce + Panel Admin básico

Objetivo:

Diseñar y desarrollar el sitio web mayorista, con su panel administrativo de productos y gestión de pedidos.

Alcance técnico:

  • Front-End (cliente):
    • Diseño personalizado con identidad visual cálida y profesional.
    • Maquetación en HTML5, CSS3, JavaScript.
    • Framework sugerido: React o Vue.
    • Sitio responsive (adaptado a celulares y tablets).

  • Back-End (servidor):
    • Desarrollo de lógica de negocio (productos, carrito, usuarios).
    • API RESTful para la conexión con el ERP (futura integración).
    • Lenguaje sugerido: Node.js con Express o Laravel (PHP).
    • Base de datos: PostgreSQL o MySQL.

  • Funcionalidades:
    • Registro/Login de clientes.
    • Catálogo de productos con filtros.
    • Carrito de pedidos.
    • Envío de pedido (sin pasarela de pago).
    • Seguimiento de estado de pedido.
    • Panel de administración básico: carga de productos, precios, fotos y categorías.
    • Notifi caciones automáticas por email.

Tiempo estimado: 5 semanas

Costo: $2.500.000 ARS

FASE 2 – Desarrollo del ERP (Sistema de Gestión Interno)

Objetivo:

Desarrollar un sistema privado para el equipo interno de la empresa, que permita administrar y centralizar la operación del negocio.

¿Qué es un ERP?

Un ERP (Enterprise Resource Planning) es un sistema que permite gestionar y automatizar áreas clave de una empresa, como stock, compras, ventas y clientes. El objetivo es reducir tareas manuales y tener control total en tiempo real.

Alcance técnico:

  • Desarrollo de un panel completo con acceso restringido para administradores, vendedores, stockistas, etc.
  • Control de permisos según roles.
  • Interfaz desarrollada en React/Vue (Front-End).
  • Backend RESTful en Node.js/Express o Laravel.
  • Base de datos conectada al sistema eCommerce.

Módulos incluidos:

  • Ventas: registro manual y desde la web, historial, comprobantes (PDF).
  • Stock: ingreso, egreso automático por ventas, alertas de stock bajo.
  • Compras: registro de compras a proveedores, actualización de stock.
  • Clientes: historial de pedidos, preferencias, cuentas corrientes.
  • Productos: carga, edición, categorías, unidades por lote.
  • Reportes: ventas, compras, productos más vendidos, stock, exportación Excel/PDF.
  • Usuarios y Roles: permisos por tipo de usuario.

Tiempo estimado: 6 semanas

Costo: $3.500.000 ARS

FASE 3 – Integración total Web + ERP (sincronización y automatización)

Objetivo:

Conectar el sitio eCommerce con el sistema ERP para automatizar procesos y evitar la carga doble de datos.

Alcance técnico:

  • Desarrollo de servicios de sincronización entre ERP y eCommerce.
  • API bidireccional entre los dos sistemas.
  • Automatización de procesos clave:
    • Venta online actualiza stock en ERP en tiempo real.
    • Cambios en productos del ERP se refl ejan en la tienda automáticamente.
    • Reportes y estadísticas se actualizan sin intervención manual.
    • Notifi caciones automáticas al cliente y al equipo interno.

Beneficios:

  • Ahorro de tiempo en tareas administrativas.
  • Reducción de errores por carga manual.
  • Visibilidad completa del estado del negocio desde un solo sistema.

Tiempo estimado: 3 semanas

Costo: $2.000.000 ARS
