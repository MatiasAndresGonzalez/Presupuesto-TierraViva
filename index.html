Descripción de la entrevista con el cliente

Cliente entrevistado: Responsable de una distribuidora mayorista de productos artesanales del país.

Objetivo de la entrevista: Relevar las necesidades específicas para el desarrollo de una página web eCommerce que se integre con su sistema de gestión actual.

Conclusión de la entrevista

El objetivo principal del sitio web es poder mostrar y vender productos artesanales al por mayor, con precios y condiciones para distribuidores.

las funcionalidades que debe tener: Catálogo dinámico de productos, carrito, pedidos, contacto, integración con el sistema de gestión, control automático de stock.

Los usuarios que utilizarán el sistema son clientes mayoristas y administradores internos.

Tenen un sistema de gestión actual que maneja stock, ventas, artículos, compras y clientes de forma interna. El sitio web debe integrarse con ese sistema.

Desean modificar los productos desde el sistema de gestión interno y la web debe reflejar los cambios automácamente.

Por ahora no requieren pasarela de pagos, solo pedidos online, el pago se coordina por transferencia o fuera del sitio.

Desean un diseño simple, cálido y profesional, con estética natural y artesanal.
